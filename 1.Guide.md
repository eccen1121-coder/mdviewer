# MDViewer: AI 에이전트 매니저를 위한 스마트 마크다운 가이드 (v1.0.6)

**MDViewer**는 AI 에이전트가 생성하는 방대한 양의 마크다운 데이터를 효율적으로 검토하고, 원본 문서의 훼손 없이 분석 메모를 남길 수 있도록 설계된 전문 도구입니다.

---

## 1. 핵심 가치 (Core Value)

단순한 텍스트 뷰어를 넘어, AI 기반 워크플로우에서 발생하는 문서들을 구조적으로 관리하고 분석하는 데 최적화되어 있습니다. 고성능 렌더링 엔진과 유연한 인터페이스를 통해 복잡한 문서 작업의 생산성을 극대화합니다.

---

## 2. 주요 기능 상세

### 🚀 고성능 마크다운 엔진
- **실시간 프리뷰**: `marked` 엔진을 사용하여 타이핑과 동시에 결과물을 확인하며, 대용량 파일에서도 지연 없는 렌더링을 제공합니다.
- **경량화된 구문 강조**: `highlight.js`를 최적화(1MB → 128KB)하여 적용, 다양한 프로그래밍 언어의 코드를 빠르고 정확하게 강조합니다.
- **맞춤형 뷰 모드**: Split(편집+미리보기), Editor Only, Preview Only 모드를 지원하며 패널의 좌우 위치를 자유롭게 변경할 수 있습니다.

### 📁 지능형 프로젝트 관리
- **멀티 탭 인터페이스**: 여러 문서를 동시에 열어두고 탭 단위로 전환할 수 있습니다. 특히 **Home 탭**에서 작성한 내용을 MD 파일로 저장하면 자동으로 새 탭으로 전환되며 Home은 초기화되는 스마트 워크플로우를 제공합니다.
- **비동기 디렉토리 탐색**: 수천 개의 파일이 포함된 폴더도 성능 저하 없이 탐색할 수 있는 비동기 트리 구조를 지원합니다.
- **강력한 전역 검색**: 파일명뿐만 아니라 문서 내부 콘텐츠까지 검색하여 해당 문맥(Context)을 즉시 확인할 수 있습니다.

### 📝 AI 분석 특화 '메모' 시스템
- **라인별 메타 노트**: 원본 마크다운 파일을 수정하지 않고도, 특정 줄(Line)에 대해 별도의 분석 메모를 남길 수 있습니다.
- **독립적 저장소**: 작성된 메모는 앱 데이터 폴더(`~/.mdviewer`)에 안전하게 별도 저장되어 데이터 무결성을 유지합니다.
- **메모 시각화**: 메모가 작성된 라인에는 노란색 아이콘이 표시되어 시각적으로 쉽게 식별할 수 있으며, 앱 재시작 후에도 안정적으로 표시됩니다.

### 🛠 개발자 편의 기능 (UX)
- **완성된 글로벌 다국어 지원 (i18n)**: 사이드바, 메모 섹션, 상태바 등 UI의 모든 텍스트와 동적 요소에 다국어 처리가 완료되어, 설정 변경 시 즉시 언어(KO/EN)가 전환됩니다.
- **컨텍스트 메뉴 강화**: 파일 트리에서 우클릭을 통해 즉시 '파일 경로 복사', '터미널 열기', 'Finder에서 보기'를 수행할 수 있습니다.
- **데이터 안전 가드**: 미저장 변경사항이 있는 경우 종료 전 확인 다이얼로그를 통해 데이터를 보호합니다.

---

## 3. 기술 스택 (Technical Stack)

- **Platform**: Electron (v33+)
- **Rendering**: Marked, Highlight.js (Optimized)
- **File System**: Node.js fs, Chokidar (File Watching)
- **Security**: DOMPurify (XSS Prevention)

---

## 4. 빌드 방법

```bash
# 의존성 설치
npm install

# 앱 실행
npm start

# macOS 배포용 패키지 빌드 (dmg 생성)
npm run build
```

## 5. 배포 및 설치 가이드 (Distribution)

타인에게 앱을 공유하거나 설치할 때는 다음 내용을 참고하세요.

### 📦 타인에게 공유하는 방법
- 빌드 후 `dist/` 폴더에 생성된 **`.dmg` 파일**만 전달하면 됩니다.
  - 예: `MDViewer-1.0.6-arm64.dmg` (Apple Silicon용)
- **주의**: `dist/mac/MDViewer.app` 파일은 빌드 과정에서 생성되는 중간 산출물입니다. `.app` 파일을 직접 전달해도 실행은 가능하지만, 사용자 설치 경험(Applications 폴더 이동 등)을 위해 `.dmg` 배포를 강력히 권장합니다.

### 💻 설치 방법
1. 전달받은 `.dmg` 파일을 더블 클릭하여 엽니다.
2. 창에 나타난 **MDViewer.app** 아이콘을 오른쪽의 **Applications(응용 프로그램)** 폴더로 드래그 앤 드롭합니다.
3. 설치가 완료되면 Launchpad나 응용 프로그램 폴더에서 앱을 실행할 수 있습니다.
